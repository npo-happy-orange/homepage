ご利用の流れ

1. 見学・面談

作業所や農作業の様子を見学していただきます。相談室にてご自身の状況、体調などをお伺いし、今後どのように取り組んでいくのかなどの打合せを行います。

2. 体験

利用をご希望される方は契約前の別の日に、体験利用を行っていただきます。一日の作業所や農作業の雰囲気を体験していただき、ご自身のペースに合わせた利用を考えます。

3. 利用契約

正式に利用へと移る際は、各種お手続きが必要です。詳しくはお問い合わせ下さい。

4. 契約後

契約後に面接を行い、希望する仕事や生活に向けてどのように取り組んでいくかを一緒に考えます。
